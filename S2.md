# 📘 Cours : Étude du signe d’une fonction

## 1️⃣ Définition

Étudier le **signe d’une fonction** ( f(x) ), c’est déterminer **où la fonction est positive, négative ou nulle** :

* ( f(x) > 0 ) → ( f ) est **positive**.
* ( f(x) < 0 ) → ( f ) est **négative**.
* ( f(x) = 0 ) → **racine** ou **zéro** de la fonction.

⚠️ Attention : il faut toujours tenir compte du **domaine de définition**.

---

## 2️⃣ Cas principaux

### A) Polynômes factorisés

Si une fonction est déjà factorisée, on peut directement utiliser un **tableau de signe**.

👉 Exemple :
[
f(x) = (x-3)(x+1)
]

* Racines : ( x=3 ) et ( x=-1 ).
* On place les racines dans l’ordre croissant sur une ligne.
* Chaque facteur change de signe en franchissant sa racine.
* On multiplie les signes pour obtenir le signe final de ( f(x) ).

**Tableau :**

[
\begin{array}{|c|c|c|c|c|}
\hline
x & -\infty & -1 & 3 & +\infty \
\hline
x-3 & - & - & 0 & + \
x+1 & - & 0 & + & + \
f(x) & + & 0 & - & + \
\hline
\end{array}
]

---

### B) Polynômes développés (non factorisés)

* **Si degré impair** :

  * Quand ( x \to +\infty ), le signe est celui du coefficient dominant.
  * Quand ( x \to -\infty ), signe opposé.

* **Si degré pair** :

  * Même signe aux deux infinis (celui du coefficient dominant).

👉 Exemple :
[
f(x) = x^2 - 4.
]
On factorise : ( f(x) = (x-2)(x+2) ).
Puis on fait un tableau de signe (voir plus haut).

---

### C) Fractions rationnelles

Si
[
f(x) = \frac{A(x)}{B(x)},
]
alors le **signe de ( f(x) )** dépend :

* du signe du numérateur ( A(x) ),
* du signe du dénominateur ( B(x) ),
* et des valeurs interdites (( B(x)=0 )).

👉 Exemple :
[
f(x) = \frac{x-1}{x+2}.
]

* Numérateur nul : ( x=1 ).
* Dénominateur nul : ( x=-2 ).
* On trace le tableau avec les deux lignes (numérateur et dénominateur).

---

### D) Racines carrées

Si ( f(x) = \sqrt{g(x)} ), alors ( f(x) \geq 0 ) **sur son domaine**.
Exemple :
[
f(x) = \sqrt{x-3}.
]

* Domaine : ( x \geq 3 ).
* Signe : toujours ( f(x) \geq 0 ).

---

### E) Logarithmes

Si ( f(x) = \ln(x) ),

* Domaine : ( x > 0 ).
* ( f(x) > 0 ) si ( x > 1 ).
* ( f(x) < 0 ) si ( 0 < x < 1 ).

---

## 3️⃣ Méthode pratique pour les élèves

1. **Trouver le domaine** (important avant tout).
2. **Chercher les racines** (où ( f(x)=0 )).
3. **Tracer une droite numérique** et placer les points particuliers (racines, valeurs interdites).
4. **Tester le signe** dans chaque intervalle (ou utiliser factorisation).
5. **Conclure** avec un tableau de signe clair.

---

## 4️⃣ Exemple récapitulatif

Soit
[
f(x) = \frac{(x-2)(x+1)}{x-3}.
]

* Domaine : ( \mathbb{R} \setminus {3} ).
* Zéros : ( x=2 ), ( x=-1 ).
* On fait un tableau :

[
\begin{array}{|c|c|c|c|c|c|}
\hline
x & -\infty & -1 & 2 & 3 & +\infty \
\hline
x-2 & - & - & 0 & - & + \
x+1 & - & 0 & + & + & + \
x-3 & - & - & - & 0 & + \
f(x) & - & 0 & + & / & + \
\hline
\end{array}
]

---

✅ Ce qu’il faut retenir :

* **Factoriser au maximum.**
* **Identifier les racines et valeurs interdites.**
* **Utiliser un tableau de signe pour être clair.**

# 📘 Rappels sur la variation d’une fonction

1. **Définition** :
   Le **tableau de variations** montre comment la fonction évolue :

* est-elle **croissante** (↗) ?
* est-elle **décroissante** (↘) ?

2. **Outil principal** :
   👉 On utilise **la dérivée f’(x)**.

* Si **f’(x) > 0** alors la fonction est **croissante**.
* Si **f’(x) < 0** alors la fonction est **décroissante**.
* Si **f’(x) = 0** alors on a un **extremum** (minimum ou maximum local).

3. **Étapes à suivre** :

* Déterminer le **domaine de définition**.
* Calculer **f’(x)**.
* Résoudre **f’(x) = 0** (points critiques).
* Étudier le signe de **f’(x)** pour chaque intervalle.
* Construire le tableau de variations.

---

## 📊 Exemple simple

Soit :
[
f(x) = x^2 - 4x + 3
]

### 1. Domaine :

(\mathbb{R}) (polynôme défini partout).

### 2. Calcul de la dérivée :

[
f’(x) = 2x - 4
]

### 3. Résolution de f’(x) = 0 :

[
2x - 4 = 0 \quad \Rightarrow \quad x = 2
]

### 4. Signe de f’(x) :

* Pour (x < 2), f’(x) < 0 → la fonction est **décroissante**.
* Pour (x > 2), f’(x) > 0 → la fonction est **croissante**.

### 5. Valeur en x = 2 :

[
f(2) = (2)^2 - 4(2) + 3 = 4 - 8 + 3 = -1
]

---

## 📋 Tableau de variations

| x     | -∞ | 2  | +∞ |
| ----- | -- | -- | -- |
| f’(x) | –  | 0  | +  |
| f(x)  | +∞ | -1 | +∞ |

👉 Conclusion :

* f(x) **décroît** de +∞ à -1 quand x → 2.
* f(x) **croît** de -1 à +∞ quand x > 2.

# 📘 Remarques graphiques (Terminale – étude de fonctions)

Quand on a fait :

1. **Domaine de définition**
2. **Limites**
3. **Dérivée et variations**
4. **Signe**

➡️ Il faut savoir **interpréter graphiquement** la fonction.

---

## 🔹 Points importants à analyser graphiquement

1. **Les zéros de f(x)**

* Ce sont les solutions de f(x) = 0.
* Graphiquement : ce sont les points d’intersection avec l’axe des abscisses (Ox).

---

2. **Les limites**

* Quand x → +∞ ou -∞, on regarde si la courbe :

  * **Monte vers +∞**
  * **Descend vers -∞**
  * Se rapproche d’une **asymptote horizontale** (exemple : y = a).

* Si la fonction a une limite infinie en un point du domaine exclu → il y a une **asymptote verticale**.

---

3. **Les variations**

* Grâce au tableau de variations, on peut tracer la **forme générale** de la courbe :

  * ↘ signifie que la courbe **descend**.
  * ↗ signifie que la courbe **monte**.
* En x où f’(x) = 0 : on a un **minimum ou maximum local** → un sommet sur la courbe.

---

4. **Le signe de f(x)**

* Si f(x) > 0, la courbe est **au-dessus de l’axe des abscisses**.
* Si f(x) < 0, la courbe est **en dessous de l’axe des abscisses**.

---

5. **Symétries éventuelles**

* Si f(-x) = f(x) → fonction **paire** → symétrie par rapport à l’axe des ordonnées.
* Si f(-x) = -f(x) → fonction **impaire** → symétrie par rapport à l’origine.

---

## 📊 Exemple rapide

Prenons :
[
f(x) = \frac{1}{x}
]

* Domaine : ℝ \ {0}
* Limites :

  * quand x → 0⁺, f(x) → +∞
  * quand x → 0⁻, f(x) → -∞
  * quand x → ±∞, f(x) → 0
* Asymptotes :

  * verticale en x = 0
  * horizontale en y = 0
* Signe :

  * positif pour x > 0
  * négatif pour x < 0
* Graphique : 2 branches, une dans le 1er quadrant et une dans le 3e.

---

👉 En résumé, à cette dernière étape, on **trace un portrait robot** de la courbe en utilisant :

* le domaine
* les limites (asymptotes)
* le signe
* les variations

# Exercice

Soit la fonction
[
f(x)=x^{3}-3x^{2}-x+3 .
]

**Objectif** : étude complète (domaine, dérivée, résolution (f'(x)=0), règle de signe de (a), tableau de variations, zéros, remarques graphiques).

---

## 1) Domaine

C'est un polynôme donc :
[
\mathcal{D}_f=\mathbb{R}.
]

---

## 2) Zéros de (f) (pour information / signe)

Factorisons (f) :
[
f(x)=x^{3}-3x^{2}-x+3=(x-3)(x-1)(x+1).
]
Donc racines de (f): (x=-1,; x=1,; x=3). La courbe coupe l’axe des abscisses en ces 3 points.

---

## 3) Dérivée

Calcul :
[
f'(x)=3x^{2}-6x-1.
]
C’est un polynôme du **2ᵉ degré** de la forme (ax^{2}+bx+c) avec ici (a=3,; b=-6,; c=-1).

---

## 4) Règle de signe du coefficient (a) (interprétation)

Pour un trinôme quadratique (ax^{2}+bx+c) :

* si (a>0) ⇒ la parabole (y=ax^{2}+bx+c) **est tournée vers le haut** ; le polynôme est **positif** pour (x) assez petits et assez grands (positif à l’extérieur des racines) et **négatif** entre les racines (si elles existent réelles) ;
* si (a<0) ⇒ la parabole **est tournée vers le bas** ; signe inverse.

Ici (a=3>0) ⇒ (f'(x)) est **>0** à l’extérieur des racines et **<0** entre les deux racines.

---

## 5) Résolution (f'(x)=0) (calcul exact)

Résolvons (3x^{2}-6x-1=0).
Formule quadratique :
[
x=\frac{6\pm\sqrt{(-6)^{2}-4\cdot3\cdot(-1)}}{2\cdot3}
=\frac{6\pm\sqrt{36+12}}{6}
=\frac{6\pm\sqrt{48}}{6}
=\frac{6\pm 4\sqrt{3}}{6}
=1\pm\frac{2\sqrt{3}}{3}.
]
Posons
[
x_{1}=1-\frac{2\sqrt{3}}{3}\quad(\approx -0{,}1547005),\qquad
x_{2}=1+\frac{2\sqrt{3}}{3}\quad(\approx 2{,}1547005).
]

---

## 6) Valeurs de (f) en ces points critiques (exactes et approximatives)

Calcul exact (remarquable) :
[
f!\Big(1\pm\frac{2\sqrt{3}}{3}\Big)=\mp\frac{16\sqrt{3}}{9}.
]
Donc :

* (f(x_{1})=+\dfrac{16\sqrt{3}}{9}\approx 3{,}0792014).
* (f(x_{2})=-\dfrac{16\sqrt{3}}{9}\approx -3{,}0792014).

(Remarque : on voit une symétrie des valeurs en signe opposé.)

---

## 7) Signe de (f'(x)) et tableau de variations

Rappel : (a=3>0) ⇒ (f') > 0 pour (x<x_{1}) et pour (x>x_{2}); (f') < 0 pour (x_{1}<x<x_{2}).

Tableau de variation (présentation simple) :

[
\begin{array}{c|ccccccc}
x & -\infty & & x_{1} & & x_{2} & & +\infty\\hline
f'(x) & + & & 0 & - & 0 & + & \
f(x) & +\infty &\searrow & \dfrac{16\sqrt{3}}{9} & \searrow & -\dfrac{16\sqrt{3}}{9} & \nearrow & +\infty
\end{array}
]

Interprétation :

* (f) **croît** sur ((-\infty,x_{1})) ; atteint un **maximum local** en (x_{1}) de valeur (+\dfrac{16\sqrt{3}}{9}).
* (f) **décroît** sur ((x_{1},x_{2})).
* (f) **croît** sur ((x_{2},+\infty)) ; atteint un **minimum local** en (x_{2}) de valeur (-\dfrac{16\sqrt{3}}{9}).

---

## 8) Signe de (f) (rapide)

Comme factorisé précédemment (f(x)=(x+1)(x-1)(x-3)), on peut faire un tableau de signe rapide (racines en (-1,1,3)).
Ordre sur la droite réelle : (-\infty < -1 < 1 < 3 < +\infty).

Tester signes donne :

* (f(x)>0) sur (]-\infty,-1[\cup ]1,3[\cup ]3,+\infty[).
* (f(x)<0) sur (]-1,1[).

---

## 9) Remarques graphiques (comment tracer)

* Intersections avec les axes :

  * (x)-intercepts en (-1,;1,;3).
  * (y)-intercept : (f(0)=3).
* Extrémums :

  * Maximum local en (x_{1}=1-\dfrac{2\sqrt{3}}{3}) de valeur (+\dfrac{16\sqrt{3}}{9}).
  * Minimum local en (x_{2}=1+\dfrac{2\sqrt{3}}{3}) de valeur (-\dfrac{16\sqrt{3}}{9}).
* Comportement aux infinis : comme (f) est de degré 3 avec coefficient dominant (+1), (f(x)\to -\infty) quand (x\to -\infty) et (f(x)\to +\infty) quand (x\to +\infty).
* Pas d’asymptote horizontale ni verticale (polynôme), la courbe passe par les 3 racines et présente la forme typique d’un cubique avec une bosse et un creux.

---

## 10) Résumé concret à donner à l’élève

1. Domaine : (\mathbb{R}).
2. (f'(x)=3x^{2}-6x-1).
3. (f'(x)=0 \iff x=1\pm \dfrac{2\sqrt{3}}{3}).
4. (a=3>0\Rightarrow) parabole de (f') tournée vers le haut → signe de (f') : (+) en dehors des racines, (-) entre elles.
5. Tableau de variations : croissance / décroissance / croissance (max local puis min local).
6. Zéros de (f) : (-1,1,3).
