## 🌐 Limites avec fonctions trigonométriques

### Limites classiques fondamentales

$$\lim_{x \to 0} \frac{\sin(x)}{x} = 1$$
> **LA** limite fondamentale à retenir absolument

$$\lim_{x \to 0} \frac{1 - \cos(x)}{x} = 0$$

$$\lim_{x \to 0} \frac{1 - \cos(x)}{x^2} = \frac{1}{2}$$

### Limites à l'infini

- $\sin(x)$ et $\cos(x)$ **n'ont pas de limite** (oscillent entre -1 et 1)
- Mais on peut avoir :
  - $\displaystyle\lim_{x \to +\infty} \frac{\sin(x)}{x} = 0$
  - $\displaystyle\lim_{x \to +\infty} \frac{\cos(x)}{x} = 0$

### Limites particulières

- $\sin(x) \to 0$ quand $x \to k\pi$ où $k \in \mathbb{Z}$
- $\cos(x) \to 0$ quand $x \to \frac{\pi}{2} + k\pi$

### Limites avec tangente

$$\tan(x) = \frac{\sin(x)}{\cos(x)}$$

- $\displaystyle\lim_{x \to \frac{\pi}{2}^-} \tan(x) = +\infty$
- $\displaystyle\lim_{x \to \frac{\pi}{2}^+} \tan(x) = -\infty$

## 📘 Étude du signe d'une fonction

### Définition

Étudier le signe de $f(x)$ c'est déterminer où la fonction est :
- **Positive** : $f(x) > 0$
- **Négative** : $f(x) < 0$
- **Nulle** : $f(x) = 0$ (racines)

⚠️ Toujours considérer le **domaine de définition**

### Cas principaux

#### A) Polynômes factorisés

**Exemple** : $f(x) = (x-3)(x+1)$

| x     | $-\infty$ | -1 | 3  | $+\infty$ |
|-------|-----------|----|----|-----------|
| $x-3$ | −         | −  | 0  | +         |
| $x+1$ | −         | 0  | +  | +         |
| $f(x)$ | +         | 0  | −  | +         |

#### B) Polynômes développés

- **Degré impair** : signes opposés aux infinis
- **Degré pair** : même signe aux deux infinis

#### C) Fractions rationnelles

$$f(x) = \frac{A(x)}{B(x)}$$

Le signe dépend de :
- Signe du numérateur $A(x)$
- Signe du dénominateur $B(x)$
- Valeurs interdites où $B(x) = 0$

#### D) Racines carrées

Si $f(x) = \sqrt{g(x)}$, alors $f(x) \geq 0$ sur son domaine

#### E) Logarithmes

Pour $f(x) = \ln(x)$ avec $x > 0$ :
- $f(x) > 0$ si $x > 1$
- $f(x) < 0$ si $0 < x < 1$

### Méthode pratique

1. **Trouver le domaine**
2. **Chercher les racines** ($f(x) = 0$)
3. **Tracer une droite numérique**
4. **Tester le signe** dans chaque intervalle
5. **Conclure** avec un tableau de signe

## 📊 Variations d'une fonction

### Principe fondamental

On utilise **la dérivée** $f'(x)$ :

| Signe de $f'(x)$ | Variation de $f$ |
|------------------|------------------|
| $f'(x) > 0$      | Croissante ↗     |
| $f'(x) < 0$      | Décroissante ↘   |
| $f'(x) = 0$      | Extremum         |

### Étapes de l'étude

1. Déterminer le **domaine de définition**
2. Calculer **$f'(x)$**
3. Résoudre **$f'(x) = 0$** (points critiques)
4. Étudier le **signe de $f'(x)$**
5. Construire le **tableau de variations**

### Exemple : $f(x) = x^2 - 4x + 3$

**Dérivée** : $f'(x) = 2x - 4$

**Point critique** : $2x - 4 = 0 \Rightarrow x = 2$

**Signe** :
- $x < 2$ : $f'(x) < 0$ → décroissante
- $x > 2$ : $f'(x) > 0$ → croissante

**Valeur minimum** : $f(2) = -1$

| x      | $-\infty$ | 2  | $+\infty$ |
|--------|-----------|----|-----------| 
| $f'(x)$ | −         | 0  | +         |
| $f(x)$  | $+\infty$ | -1 | $+\infty$ |

## 🎨 Remarques graphiques

### Points clés à analyser

#### 1. Zéros de $f(x)$
Intersections avec l'axe des abscisses (Ox)

#### 2. Limites
- Comportement à $\pm\infty$
- **Asymptote horizontale** : $y = a$
- **Asymptote verticale** : en un point exclu du domaine

#### 3. Variations
- ↘ : courbe descend
- ↗ : courbe monte
- Extremums aux points où $f'(x) = 0$

#### 4. Signe de $f(x)$
- $f(x) > 0$ : courbe au-dessus de l'axe
- $f(x) < 0$ : courbe en-dessous de l'axe

#### 5. Symétries
- **Fonction paire** : $f(-x) = f(x)$ → symétrie par rapport à l'axe Oy
- **Fonction impaire** : $f(-x) = -f(x)$ → symétrie par rapport à l'origine

## 📝 Exercices corrigés

### Exercice 1 : Fonction polynomiale

**Énoncé** : Étude complète de $f(x) = x^3 - 3x^2 - x + 3$

<details>
<summary>Voir la correction</summary>

#### 1) Domaine
$\mathcal{D}_f = \mathbb{R}$ (polynôme)

#### 2) Factorisation et zéros
$$f(x) = (x-3)(x-1)(x+1)$$
**Racines** : $x = -1, 1, 3$

#### 3) Dérivée
$$f'(x) = 3x^2 - 6x - 1$$

#### 4) Points critiques
$$f'(x) = 0 \iff x = 1 \pm \frac{2\sqrt{3}}{3}$$

- $x_1 = 1 - \frac{2\sqrt{3}}{3} \approx -0,155$
- $x_2 = 1 + \frac{2\sqrt{3}}{3} \approx 2,155$

#### 5) Valeurs aux extremums
$$f(x_1) = \frac{16\sqrt{3}}{9} \approx 3,08 \quad \text{(maximum local)}$$
$$f(x_2) = -\frac{16\sqrt{3}}{9} \approx -3,08 \quad \text{(minimum local)}$$

#### 6) Tableau de variations

| x      | $-\infty$ | $x_1$ | $x_2$ | $+\infty$ |
|--------|-----------|-------|-------|-----------|
| $f'(x)$ | +         | 0     | −     | 0         | +     |
| $f(x)$  | $-\infty$ | ↗ | $\frac{16\sqrt{3}}{9}$ | ↘ | $-\frac{16\sqrt{3}}{9}$ | ↗ | $+\infty$ |

#### 7) Signe de $f$
- $f(x) > 0$ sur $]-\infty, -1[ \cup ]1, 3[$
- $f(x) < 0$ sur $]-1, 1[ \cup ]3, +\infty[$

</details>

### Exercice 2 : Fonction rationnelle

**Énoncé** : Étude complète de $\displaystyle f(x) = \frac{x^2 - 2x - 3}{x - 1}$

<details>
<summary>Voir la correction</summary>

#### 1) Domaine
$$\boxed{\mathcal{D}_f = \mathbb{R} \setminus \{1\}}$$

#### 2) Forme simplifiée (division euclidienne)
$$f(x) = x - 1 - \frac{4}{x-1}$$

#### 3) Limites et asymptotes

**En $x = 1$** :
- $\displaystyle\lim_{x \to 1^-} f(x) = +\infty$
- $\displaystyle\lim_{x \to 1^+} f(x) = -\infty$

→ **Asymptote verticale** : $x = 1$

**À l'infini** :
$$\lim_{x \to \pm\infty} [f(x) - (x-1)] = 0$$

→ **Asymptote oblique** : $y = x - 1$

#### 4) Zéros
$$x^2 - 2x - 3 = (x-3)(x+1) = 0$$
**Racines** : $x = -1$ et $x = 3$

#### 5) Dérivée
$$f'(x) = \frac{x^2 - 2x + 5}{(x-1)^2}$$

**Analyse** :
- Dénominateur : $(x-1)^2 > 0$ pour tout $x \neq 1$
- Numérateur : $\Delta = 4 - 20 = -16 < 0$ donc toujours positif

$$\boxed{f'(x) > 0 \text{ pour tout } x \neq 1}$$

→ **Fonction strictement croissante** sur chaque intervalle de définition

#### 6) Signe de $f$

| Intervalle          | Signe  |
|--------------------|--------|
| $]-\infty, -1[$    | −      |
| $x = -1$           | 0      |
| $]-1, 1[$          | +      |
| $x = 1$            | ∄      |
| $]1, 3[$           | −      |
| $x = 3$            | 0      |
| $]3, +\infty[$     | +      |

#### 7) Tableau de variations

```
x        : -∞     -1    1      3     +∞
f'(x)    :  +      +   (∄)     +      +
f(x)     : -∞ ↗  0 ↗ +∞ | -∞ ↗ 0 ↗  +∞
```

#### 8) Remarques graphiques

- **Asymptote verticale** : $x = 1$
- **Asymptote oblique** : $y = x - 1$
- **Points d'intersection avec Ox** : $(-1, 0)$ et $(3, 0)$
- **Point d'intersection avec Oy** : $(0, 3)$
- Deux branches monotones croissantes séparées par l'asymptote verticale

</details>

## 💡 Conseils méthodologiques

### Pour réussir une étude de fonction

1. ✅ Toujours commencer par le **domaine**
2. ✅ Factoriser au maximum
3. ✅ Utiliser les tableaux de signe
4. ✅ Vérifier la cohérence graphique
5. ✅ Calculer quelques valeurs numériques pour contrôle

### Erreurs fréquentes à éviter

- ❌ Oublier le domaine de définition
- ❌ Ne pas identifier les asymptotes
- ❌ Confondre signe de $f$ et signe de $f'$
- ❌ Oublier de tester le signe dans chaque intervalle
